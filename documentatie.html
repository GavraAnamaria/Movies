Documentatie

Pagina principala, numita "index.html" are rolul de a fi meniul aplicatiei, din care se poate naviga mai usor, aceasta duce la cerintele aplicatiei.
Ca sa ajungi dintr-o destinatie in alta, s-au folosit butoane care ajuta comunicarea dintre paginile html si php, fiind realizate cu ajutorul CSS-ului (Cascading Style Sheet).
De ce am folosit CSS?
Deoarece este mai usor sa modific intr-o singura parte design-ul pentru toata pagina, acest lucru fiind in atributul style. Astfel, daca doream sa modific toate butoanele din culoarea rosie in verde, era de ajuns sa modific intr-un singur loc, nu in toate butoane.
De ce am folosit PHP?
Pentru a face legatura dintre baza de date online, in phpmyadmin si site-ul web. Este foarte usor de folosit, fiind simplu si poate fi folosit in combinatie cu html.
De ce HTML?
HTML-ul este gasit in majoritatea aplicatilor web din ziua de azi, fiind folosit inca de la inceputurile internetului. Este usor de folosit, prin headere cum ar fi < html > < / html>, < br>(rand nou) etc. In combinatie cu CSS-ul se pot face site-uri simple si dragute.
Acest site a fost realizat cu ajutorul:
-phpmyadmin = unde am introdus codul mysql si am facut o baza de date locala, avand localhost-ul cu portul 7777, dar am verificat si interogarile daca functioneaza
-xampp = datorita careia am putut face localhost-ul
-w3schools = de un am luat codurile PHP si CSS pentru interogari, respectiv design, avand tutoriale pe acest subiect
-youtube = pentru a afla cum se face o conectare intre phpmyadmin si aplicatia web
-stackflow = pentru erorile intampinate in cod
